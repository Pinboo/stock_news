# 常见问题解决方案

## 数据获取问题

### ❌ Connection aborted / Remote end closed connection

**错误信息：**
```
获取大盘数据失败: ('Connection aborted.', RemoteDisconnected('Remote end closed connection without response'))
```

**原因：**
1. 请求频率过高，被服务器限制
2. 网络不稳定
3. 数据源（东方财富等）临时故障
4. 防火墙或代理问题

**解决方案：**

✅ **已自动优化**
- 系统已添加自动重试机制（最多3次）
- 添加了随机延迟，避免请求过快
- 失败时使用备用数据，不会中断程序

✅ **手动优化**

1. **检查网络连接**
   ```bash
   ping www.baidu.com
   ```

2. **更新 akshare**
   ```bash
   pip install --upgrade akshare
   ```

3. **修改重试参数**
   
   编辑 `stock_data.py`，增加重试次数：
   ```python
   self.max_retries = 5  # 改为5次
   self.retry_delay = 3  # 延迟改为3秒
   ```

4. **使用代理（如果在公司网络）**
   
   编辑 `stock_data.py`，添加代理设置：
   ```python
   import os
   os.environ['HTTP_PROXY'] = 'http://proxy.company.com:8080'
   os.environ['HTTPS_PROXY'] = 'http://proxy.company.com:8080'
   ```

5. **错峰使用**
   - 避开交易时间高峰（9:30-11:30, 13:00-15:00）
   - 在非交易时间测试

## AI 分析问题

### ❌ AI分析失败

**错误信息：**
```
AI分析失败: API key错误 / 余额不足 / 网络超时
```

**解决方案：**

1. **检查 API Key**
   ```bash
   python check_config.py
   ```
   确认 `OPENAI_API_KEY` 已正确配置

2. **检查账户余额**
   - OpenAI: https://platform.openai.com/account/billing
   - 通义千问: https://dashscope.console.aliyun.com/
   - 腾讯混元: https://console.cloud.tencent.com/hunyuan

3. **切换模型**
   
   编辑 `.env`，使用更便宜的模型：
   ```env
   MODEL_NAME=hunyuan-lite  # 或 deepseek-chat
   ```

4. **增加超时时间**
   
   编辑 `ai_analyzer.py`，增加 timeout：
   ```python
   response = self.client.chat.completions.create(
       model=config.MODEL_NAME,
       messages=[...],
       timeout=60  # 增加到60秒
   )
   ```

## 钉钉推送问题

### ❌ 钉钉消息发送失败

**错误信息：**
```
钉钉消息发送失败: errcode: 310000
```

**原因：**
1. Webhook 地址错误
2. 关键词不匹配
3. 签名验证失败
4. IP 不在白名单

**解决方案：**

1. **检查配置**
   ```bash
   python check_config.py
   ```

2. **测试钉钉**
   ```bash
   python test_dingtalk.py
   ```

3. **检查关键词**
   - 系统消息已包含"A股"关键词
   - 确认机器人设置的关键词是"A股"

4. **重新创建机器人**
   - 删除旧机器人
   - 重新添加，复制新的 Webhook

## Stream 模式问题

### ❌ 聊天机器人连接失败

**错误信息：**
```
连接失败: AppKey 或 AppSecret 错误
```

**解决方案：**

1. **确认已开通 Stream 模式**
   - 访问 https://open-dev.dingtalk.com/
   - 应用开发 → 开发配置 → Stream 模式 → 开启

2. **检查配置**
   ```bash
   python check_config.py
   ```
   确认 AppKey、AppSecret、AgentId 正确

3. **重新获取密钥**
   - 在开发者后台重置密钥
   - 更新 `.env` 文件

4. **检查网络**
   ```bash
   ping open.dingtalk.com
   ```

5. **查看详细日志**
   
   编辑 `dingtalk_chat_bot.py`，启用调试：
   ```python
   logging.basicConfig(level=logging.DEBUG)
   ```

## 性能优化

### 🐌 运行速度慢

**优化方案：**

1. **减少股票数量**
   
   编辑 `.env`：
   ```env
   STOCK_POOL=000001,600519  # 只关注2只
   ```

2. **降低推送频率**
   
   编辑 `.env`：
   ```env
   PUSH_TIME=15:30  # 只在收盘后推送
   ```

3. **使用更快的模型**
   
   编辑 `.env`：
   ```env
   MODEL_NAME=hunyuan-turbo  # 或 qwen-turbo
   ```

4. **并发获取数据**
   
   编辑 `main.py`，使用线程池：
   ```python
   from concurrent.futures import ThreadPoolExecutor
   
   with ThreadPoolExecutor(max_workers=3) as executor:
       futures = [executor.submit(get_stock, code) for code in codes]
   ```

## 紧急处理

### 🚨 系统完全无法运行

**快速恢复：**

1. **备份配置**
   ```bash
   copy .env .env.backup
   ```

2. **重新安装**
   ```bash
   pip uninstall -y akshare requests openai
   pip install -r requirements.txt
   ```

3. **使用最小配置**
   
   编辑 `.env`：
   ```env
   # 只配置必需项
   DINGTALK_APPKEY=你的AppKey
   DINGTALK_APPSECRET=你的AppSecret
   DINGTALK_AGENT_ID=你的AgentId
   OPENAI_API_KEY=你的APIKey
   OPENAI_BASE_URL=https://api.hunyuan.cloud.tencent.com/v1
   MODEL_NAME=hunyuan-lite
   STOCK_POOL=000001
   PUSH_TIME=15:30
   ```

4. **逐步测试**
   ```bash
   python check_config.py      # 检查配置
   python test_dingtalk.py     # 测试钉钉
   python test_run.py          # 测试功能
   ```
