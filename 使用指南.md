# 🚀 使用指南

## 快速开始

### 方式一：一键启动（推荐）⭐

双击运行 `一键启动全部功能.bat`

会自动打开两个窗口：
- 窗口1：定时推送服务
- 窗口2：聊天机器人服务

### 方式二：分别启动

根据需要选择启动：

1. **只要定时推送**
   - 双击 `4-启动定时任务.bat`
   - 每天 09:00 和 15:30 自动推送

2. **只要聊天机器人**
   - 双击 `6-启动聊天机器人.bat`
   - 在钉钉群里@机器人聊天

3. **两个都要**
   - 双击 `一键启动全部功能.bat`
   - 或分别运行上面两个

## 功能说明

### 📊 定时推送服务

**功能：**
- 每天定时推送股市分析报告
- 包含：新闻、大盘、个股、AI分析

**推送时间：**
- 09:00（开盘后）
- 15:30（收盘后）

**修改推送时间：**
编辑 `.env` 文件：
```env
PUSH_TIME=09:00,12:00,15:30
```

---

### 💬 聊天机器人服务

**功能：**
- 在钉钉群里@机器人实时问答
- 查询股票、大盘、新闻

**使用方法：**
```
@机器人 000001      # 查询平安银行
@机器人 大盘        # 查询大盘行情
@机器人 新闻        # 查询财经新闻
@机器人 帮助        # 查看帮助
```

**前提条件：**
- 需要在钉钉开发者后台开通 Stream 模式
- 详见：`钉钉Stream模式配置.md`

## 管理工具

### 查看运行状态
双击 `查看运行状态.bat`

显示哪些服务正在运行

### 停止服务

**方法一：关闭窗口**
- 直接关闭对应的命令行窗口

**方法二：按 Ctrl+C**
- 在命令行窗口中按 Ctrl+C

**方法三：任务管理器**
1. 按 Ctrl+Shift+Esc
2. 找到 Python 进程
3. 右键 → 结束任务

## 常见问题

### Q1: 如何同时运行两个服务？
A: 双击 `一键启动全部功能.bat`

### Q2: 如何只运行定时推送？
A: 双击 `4-启动定时任务.bat`

### Q3: 如何只运行聊天机器人？
A: 双击 `6-启动聊天机器人.bat`

### Q4: 如何查看服务是否在运行？
A: 双击 `查看运行状态.bat`

### Q5: 如何停止服务？
A: 关闭对应的命令行窗口，或按 Ctrl+C

### Q6: 关闭电脑后服务还运行吗？
A: 不会。需要电脑开机且程序运行中。

### Q7: 可以在后台运行吗？
A: 可以最小化窗口，但不能关闭。

### Q8: 如何修改推送时间？
A: 编辑 `.env` 文件中的 `PUSH_TIME`

### Q9: 如何修改关注的股票？
A: 编辑 `.env` 文件中的 `STOCK_POOL`

### Q10: 聊天机器人不回复？
A: 
1. 检查程序是否在运行
2. 确认已开通 Stream 模式
3. 查看命令行窗口的错误信息
